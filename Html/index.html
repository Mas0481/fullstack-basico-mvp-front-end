MVP - Desenvolvimento Full Stack Aluno: Marcio Almeida da Silva

Sistema de cadastro de colaboradores desenvolvido como MVP da trilha de Desenvolvimento Full Stack. O projeto implementa um CRUD completo (Create, Read, Update e Delete) para gerenciamento de colaboradores, utilizando uma API REST construída com Flask e uma interface web responsiva para interação com o usuário. Além das funcionalidades de cadastro, consulta, edição e exclusão de registros, a aplicação possui documentação da API gerada com Swagger/OpenAPI, proporcionando uma descrição detalhada dos endpoints, parâmetros, respostas e exemplos de requisições, facilitando o desenvolvimento, testes e futuras integrações.
