Frontend - Cadastro de Funcionários

Como executar

Crie uma pasta para seu projeto.

Faça o download do projeto Html dentro da pasta do projeto.

Abrir o arquivo index.html no seu browser.

Observação Importante:
É vital para o funcionamento do html que a API esteja rodando.


Para acessar a documentação completa acesse

http://127.0.0.1:5000/docs

Você verá a tela interativa do Swagger com todas as rotas prontas, documentadas e com o botão "Try it out" para testar as requisições ali mesmo.
